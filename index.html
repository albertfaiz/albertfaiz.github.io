Portfolio

Below are some of my key projects, showcasing expertise in data science, machine learning, and geospatial analysis using tools like Python, Geopandas, and Rasterio.

Crop Yield Prediction

Developed a machine learning model to predict crop yields in Dallas County using gNATSGO soil data and Random Forest algorithms.

Livestock Data Analysis

Processed FAO GLW data to compute livestock density and growth rates across U.S. counties using Geopandas and Rasterio.

Utilized SHAP to interpret feature importance in machine learning models for life expectancy prediction.
